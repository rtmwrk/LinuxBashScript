### "[get_proc_del_file.sh](./get_proc_del_file.sh)" - мониторинг процессов, кот. продолжают удерживать удаленные файлы

---

Скрипт выводит список процессов, кот. продолжают удерживать удаленные, но продолжающие занимать дисковое пространство, файлы.

Скрипт был подготовлен для работы в условиях **"РЕД ОС"** версий 7.3, 8.0.
